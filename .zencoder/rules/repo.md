# hiero-sdk-python repository notes

- **Language**: Python
- **Package layout**: source code under `src/hiero_sdk_python`
- **Testing**: `pytest` (unit tests under `tests/unit`, integration tests under `tests/integration`)
- **Tooling**: `pyproject.toml` (Poetry-compatible), `mypy.ini` for static typing
- **Docs**: README at root, `docs/` for additional documentation, `examples/` for sample scripts
